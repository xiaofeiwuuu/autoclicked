## ✨ 功能特性

### 🖱️ 点击控制
- **鼠标按键**: 左键 / 右键 / 中键
- **点击类型**: 单击 / 双击 / 三击
- **点击间隔**: 毫秒 / 秒 / 分钟
- **点击次数**: 指定次数或无限循环（0=无限）
- **延迟启动**: 倒计时后自动启动

### 📍 点击模式
1. **跟随鼠标** - 在当前鼠标位置点击
2. **固定位置** - 在指定坐标重复点击
3. **随机区域** - 在矩形区域内随机点击

### 🎯 人性化模拟
- **延迟波动**: 0-100% 随机间隔变化，模拟真人节奏
- **坐标抖动**: 0-10px 点击位置偏移，避免完全相同坐标

### ⌨️ 全局快捷键
- **启动**: `⌘ Cmd + ⌥ Opt + S`
- **停止**: `⌘ Cmd + ⌥ Opt + X`
- **捕获坐标**: `⌘ Cmd + ⌥ Opt + C`

### 🌍 多语言
简体中文 / 繁体中文 / English / 日本語 / 한국어 / Français / Deutsch / Español

### 🎨 主题
跟随系统 / 浅色 / 深色
